# FAA Store - ICT Project

A clean and responsive front-end e-commerce website featuring clothing, accessories, fragrances, and scarfs.

## How to Run Locally

You can run the project locally using one of the following methods:

### Method 1: Double-Click (Simple)
Simply open the folder and double-click **`home.html`** to open it directly in your web browser.

### Method 2: Local Web Server (Recommended)
Running a local web server ensures all paths and browser features work correctly.

1. Open your terminal in this project directory.
2. Start a local server:
   - **If you have Python installed**:
     ```bash
     python3 -m http.server 8000
     ```
   - **If you have Node.js / npm installed**:
     ```bash
     npx serve .
     ```
3. Open the following local link in your browser:
   👉 **[http://localhost:8000/home.html](http://localhost:8000/home.html)** (or the port specified by `serve`).
